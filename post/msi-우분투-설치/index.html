MSI노트북에 우분투 18.04 설치하기

2018-09-10
설치

Contents

        • BIOS 설정
        • 윈도우 설치
        • 우분투 설치 (참조)

처음에 MSI GS63 7RD Stealth 노트북에 우분투 듀얼 부팅했을 때 멈춤현상이 너무 심했다. 아래는 여러 삽질 끝에 해결한 과정이다. 왜 저런 방법으로 설정을 해야하는지는 잘 모른다. 우분투를 재설치하면 BIOS와 우분투 설치 과정에 이미지 추가하도록 하겠다.

BIOS 설정

  • UEFI 모드
  • Fast Boot 끄기
  • Secure Boot 끄기
  • Speed step 끄기
  • 부팅 순서 바꾸기
  • 저장 및 재시작


윈도우 설치

  • 위 BIOS 설정을 유지한 상태로 윈도우 설치
  • diskmgmt.msc 를 입력하여 디스크 관리에 들어감
  • 용량을 적절히 나눔
  • 전원 관리 옵션 선택에 들어감
  • 전원 단추 작동 설정을 누름
  • 현재 사용할 수 없는 설정 변경
  • 종료 설정에서 빠른 시작 켜기 해제


우분투 설치 (참조)

  • 하이라이트 표시 부분을 Try ubuntu without installing 위에 두고 E 키를 누르기
  • 리눅스 줄 끝 --- 뒤에 한 칸 띄고 nouveau.modeset=0 을 적고 F10 누르기
  • 8192 (노트북 램 8gb), 논리, 시작, SWAP
  • 나머지 용량 EX4, / 설정
  • 설치가 끝나고 재시작할 때 USB 뽑지 않기
  • BIOS에 들어가서 부팅 순서를 다시 원래대로 바꿔 주고 USB 제거
  • 부팅 순서 아래에 있는 ‘UEFI Hard Disk Drive BBS Priorities’에 들어감
  • Boot option1을 ubuntu로 바꾸고 재시작

  • GNU GRUB라고 아래처럼 나오면 하이라이를 Ubuntu에 두고 E 키를 누름

    • Ubuntu
    • Advanced options for Ubuntu
    • …

  • 리눅스 줄 마지막에 nouveau.modeset=0 적고 F10을 누르면 끝
